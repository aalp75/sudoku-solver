## Pedantix
Execute the first cells, then:

1) Choose a language (french: fr, english: en),
2) Choose the game number (-1 for the game of the day),
3) Execute last cell,
4) New game starts at 12:00 for pedantix (FR) and 20:00 for pedantle(EN), UTC+2.
